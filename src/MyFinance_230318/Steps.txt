1. Найти необходимые иконки
    - Иконки можно бесплатно брать в формате png с сайта https://www.flaticon.com/
    - Также бесплатно можно брать иконки https://www.iconfinder.com/
    - Онлайн редактор для иконок https://iconsflow.com/
    - Загрузить их в папку Icons

2. Найти и скачать необходимые шрифты
   - Скачать с сайта https://fonts.google.com/ или другого
   - Загрузить их в папку Fonts

3. В основном пакете MyFinance_230318 создать подпакет MyFinance_230318.settings
4. В подпакете MyFinance_230318.settings создать class Texts, в котором константы будут храниться в виде HashMap
    - Создается переменная data и импортируется java.util.HashMap
5. Создаем инициализирующий метод init() для заполнения константами, заполняем значениями ключ-значение
6. Создаем метод metodGet(String key) получающий значение по ключу
7. Создаем метод возвращающий массив месяцев

8. Не удалось подключить шрифты (уточнить куда размещать папку фонтс, изучить исключения)

9. Создаем в пакете MyFinance_230318 новый подпакет MyFinance_230318.models в котором будет находится абстрактный класс Commons.java
